#+TITLE: crontab 格式

* 
| 分 | 小时     | 天(月) | 月 | 天（周） | 用户 | 命令                                        |
|----+----------+--------+----+----------+------+---------------------------------------------|
| 17 | *        | *      | *  | *        | root | cd / && run-parts --report /etc/cron.hourly |
| 17 | */1      | *      | *  | *        | root | cd / && run-parts --report /etc/cron.hourly |
| 17 | [0-59]/1 | *      | *  | *        | root | cd / && run-parts --report /etc/cron.hourly |
|    |          |        |    |          |      |                                             |

* 参数格式
  - 分钟: 0-59
  - 小时: 0-23
  - 天（月）: 1-31
  - 月: 1-12
  - 天（周）: 0~6，0为星期天
